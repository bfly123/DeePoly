## 收敛阶计算说明

收敛阶 p 的计算公式: p = log(e1/e2) / log(h1/h2)

其中:
- e1, e2 是相邻两个网格的误差
- h1, h2 是相邻两个网格的步长
- 理论上对于二阶精度方法，收敛阶应该接近2
- **DeePoly方法的收敛阶远超理论值，表明具有超高阶精度特性**

## 结论

1. **超高精度**: DeePoly达到了机器精度级别的误差（10^-21量级）
2. **超收敛性**: 平均收敛阶远超传统二阶方法的理论值
3. **计算效率**: 大部分时间用于神经网络训练，方程拟合非常快速
4. **稳定性**: 误差随着分段数增加呈现稳定的递减趋势
